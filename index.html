Introduction

What is java?

Java is a high-level, object-oriented, and class-based programming language designed to run on multiple devices and operating systems. It was originally developed by James Gosling and his team at Sun Microsystems in 1995. Today, Java is maintained by Oracle Corporation.

Java it is used for:

  • Mobile apps
  • Web apps
  • Desktop apps
  • Games
  • Database connection
  • Cloud-based services
Variables

A variable is a container for storing data values. In Java, each variable must be declared with a specific data type, which determines the kind of data it can hold. The syntax for declaring a variable in Java is as follows: 

          dataType variableName = value;

Example:


          int age = 69;
          double height = 1.89;
          boolean isStudent = true;
          char gender = 'M';

In this example, the variables age, height, isStudent, and gender are declared with the appropriate data types and assigned values.

Java Syntax

What is Java syntax?

Java syntax is the set of rules that define a correctly structured Java program. It includes the use of keywords, operators, and punctuation to create valid statements and expressions. Some key elements of Java syntax include:

  • Case sensitivity: Java is case-sensitive, meaning that variable names and keywords must be used with the correct capitalization.
  • Semicolons: Each statement in Java must end with a semicolon.
  • Braces: Curly braces ({}) are used to define code blocks, such as classes and methods.
  • Comments: Single-line comments start with //, while multi-line comments are enclosed in /* */.

Example:


          // This is a single-line comment
          /* This is a multi-line comment */
          public class HelloWorld {
            public static void main(String[] args) {
              System.out.println("Hello, World!");
            }
          }
Operators in Java

What are operators?

Operators are special symbols that perform operations on variables and values. Java supports various types of operators, including:

  • Arithmetic operators: +, -, *, /, %
  • Relational operators: ==, !=, >, <, >=, <=
  • Logical operators: &&, ||, !
  • Assignment operators: =, +=, -=, *=, /=
  • Unary operators: ++, --

Example:


          int a = 10;
          int b = 5;
          int sum = a + b; // sum = 15
          boolean isEqual = (a == b); // isEqual = false
          boolean isTrue = (a > b) && (b < 10); // isTrue = true

In this example, various operators are used to perform arithmetic and logical operations on the variables a and b.

Non Primitive Data Types

Non-primitive data types are data types that are not defined by the Java programming language. They are created using primitive data types and can store multiple values. Some common non-primitive data types in Java include:

  • Strings: A sequence of characters.
  • Arrays: A collection of elements of the same type.
  • Classes: A blueprint for creating objects.
  • Interfaces: A contract that defines a set of methods.
  • Enums: A special data type that enables a variable to be a set of predefined constants.

Example:


          String name = "galuh";
          System.out.println(name.toUpperCase()); // Output: GALUH

In this example, the String class is used to create a string object named name, and the toUpperCase() method is called to convert the string to uppercase.

Inputs and Outputs

Inputs and outputs refer to the process of receiving data from the user and displaying results. In Java, the Scanner class is commonly used for input, while the System.out class is used for output.

Example:


          import java.util.Scanner;
          public class InputOutputExample {
            public static void main(String[] args) {
              Scanner scanner = new Scanner(System.in);
              System.out.print("Enter your name: ");
              String name = scanner.nextLine();
              System.out.println("Hello, " + name + "!");
              scanner.close();
            }
          }

In this example, the Scanner class is used to read user input from the console, and the System.out.println() method is used to display a greeting message.

Conditional Statements

Conditional statements are used to perform different actions based on different conditions. In Java, the most common conditional statements are if, else if, and else. The switch statement is also used for multiple conditions.

Example:


          int age = 18;
          if (age >= 18) {
            System.out.println("You are an adult.");
          } else {
            System.out.println("You are a minor.");
          }

In this example, the if statement checks whether the variable age is greater than or equal to 18 and prints the appropriate message.

Loops

Loops are used to execute a block of code repeatedly until a specified condition is met. In Java, the most common types of loops are for, while, and do-while loops.

Example:


          for (int i = 0; i < 5; i++) {
            System.out.println("Iteration: " + i);
          }
          /* Output:  Iteration: 0
                      Iteration: 1  
                      Iteration: 2
                      Iteration: 3
                      Iteration: 4
          */
          //  The for loop iterates five times, printing the current iteration number.

In this example, the for loop iterates five times, printing the current iteration number.

Methods

Methods are blocks of code that perform a specific task and can be called from other parts of the program. They help organize code and promote reusability. In Java, methods can have parameters and return values.

Example:


          public class MethodExample {
            public static void main(String[] args) {
              int result = add(5, 10);
              System.out.println("Sum: " + result); // Output: Sum: 15
            }
            public static int add(int a, int b) {
              return a + b;
            }
          }

In this example, the add() method takes two integer parameters and returns their sum.

Introduction to OOP

What is Object-Oriented Programming (OOP)?

Object-Oriented Programming (OOP) is a programming paradigm that uses objects to represent data and methods. Java is an object-oriented language, which means it supports the principles of OOP, including:

  • Encapsulation: Bundling data and methods into a single unit.
  • Inheritance: Creating new classes based on existing ones.
  • Polymorphism: Using a single interface to represent different data types.
  • Abstraction: Hiding complex implementation details from the user.

Example:


          public class Car {
            String color = "Red";
        
            void drive() {
                System.out.println("The car is driving");
            }
        }
        
        public class Main {
            public static void main(String[] args) {
                Car c = new Car();
                System.out.println(c.color);
                c.drive();
            }
        }

In this example, the Car class represents a car with a color and a drive() method. The Main class creates an instance of the Car class and calls the drive() method.
